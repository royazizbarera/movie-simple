# Movie Simple

## Run
``` bash
  $env:PORT=3002
  npm start
```


## Install
Font
``` bash
npm i @fontsource/quicksand
npm i @fontsource/roboto
```
React Router Dom
``` bash
npm i react-router-dom
```
All
``` bash
npm i axios
npm i @mui/joy @emotion/react @emotion/styled @mui/icons-material @mui/material
npm i query-string

```


## Usage
Usage Font
``` bash
import "@fontsource/quicksand"
import "@fontsource/roboto"
```
